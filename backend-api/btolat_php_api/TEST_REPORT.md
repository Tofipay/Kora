# تقرير اختبار Btolat Videos JSON API

**تاريخ الاختبار:** 12 يوليو 2026  
**البيئة:** PHP 8.3.6 على Ubuntu 24.04

## الملخص

اجتاز المشروع فحص الصياغة لجميع ملفات PHP وأربعة اختبارات تكامل مباشرة مع الموقع المصدر. كما اختُبرت واجهة HTTP الفعلية لقسم الكل، والتصنيفات، وكأس العالم، وحالات الخطأ، وجمع عشر صفحات متتابعة.

| الاختبار | النتيجة |
|---|---|
| فحص صياغة جميع ملفات PHP | ناجح، دون أخطاء |
| الصفحة الأولى من قسم الكل | ناجح |
| الصفحة الثانية عبر مؤشّر التحميل | ناجح، دون تداخل مع الصفحة الأولى |
| تصنيف كأس العالم | ناجح، 15 عنصرًا في العينة |
| تفاصيل فيديو مفرد | ناجح |
| نقطة فحص الخدمة | JSON صالح |
| قائمة التصنيفات | JSON صالح، 25 تصنيفًا |
| تصنيف غير موجود | HTTP 400 وJSON خطأ صالح |
| جمع الصفحات 1 إلى 10 | ناجح |
| الوصول المباشر إلى الصفحة 10 | ناجح |

## نتيجة اختبار الصفحات المتعددة

أعاد الطلب `category=all&page=1&pages=10` عشر دفعات فعلية بإجمالي **150 فيديو فريدًا** دون تكرار، وأعاد `next_page=11`. كما أعاد طلب `category=all&page=10&pages=1` عدد **15 فيديو**، مما يثبت دعم الصفحات العاشرة وما بعدها عبر محاكاة مؤشّر المصدر.

## أوامر التحقق

```bash
php tests/run.php
php -S 127.0.0.1:8080 -t public
curl -s 'http://127.0.0.1:8080/?endpoint=videos&category=all&page=1&pages=10'
```

## تنبيه الصيانة

الاختبارات تثبت توافق المشروع مع بنية الموقع وقت التسليم. لأن المصدر صفحة ويب خارجية وليس عقد API رسميًا معلنًا، يجب إعادة تشغيل الاختبارات بعد أي تغيير ملحوظ في تصميم بطاقات الموقع أو آلية التحميل.
