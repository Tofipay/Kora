# تقرير تحديث ALOKA Live — الإصدار V5

## ما تمت إضافته

- بحث داخل إدارة الأخبار بواسطة رقم الخبر أو رابط الخبر الكامل.
- استخراج رقم الخبر تلقائياً من الرابط العربي المشفر، مثل `14006412`.
- فتح الخبر المحلي أو خبر الـAPI مباشرة داخل نموذج التعديل.
- تحويل إدارة كتالوج القنوات إلى ثلاث صفحات مستقلة:
  1. أقسام القنوات.
  2. قوائم القنوات داخل القسم.
  3. القنوات داخل القائمة.
- الضغط العادي على القسم أو القائمة يفتح الصفحة التالية.
- الضغط المطوّل على بطاقات الأدمن يعرض «تعديل / حذف».
- الأقسام لا تستخدم صوراً، وحقولها الظاهرة هي الاسم العربي والإنجليزي فقط.
- قائمة القنوات تستخدم الصورة والاسم العربي والإنجليزي والترتيب فقط.
- القناة تستخدم الاسم العربي والإنجليزي واللوقو والرابط/الكود والترتيب فقط.
- تفعيل دعم المفضلة تلقائياً لكل قناة جديدة.
- إعادة تصميم صفحة القنوات إلى أقسام مع سلايد أفقي احترافي.
- إعادة تصميم صفحة قنوات المجموعة إلى شبكة لوقوهات من ثلاثة أعمدة على الجوال.
- الضغط على القناة يفتح دائماً Intent رسمي بالحزمة `com.aloka.live.app` والمخطط `xmtv`.
- عند تعذر فتح التطبيق يظهر Dialog عربي/إنجليزي، والتحميل من `https://t.me/alokalive` فقط.
- منع التحديد والنسخ وقائمة الضغط المطوّل والمشاركة من صفحات الموقع العامة، مع إبقاء حقول الإدخال قابلة للاستخدام.

## ما لم يتغير

- `User-Agent: com.aloka.live.app`.
- `AppChannels` و`ChannelLib` و`Streams` وروابط تشغيل المباريات.
- مسارات API الحالية.
- الأخبار الأصلية وطريقة جلبها.
- الأفلام والمسلسلات والمباريات والمفضلة وPWA وSEO واللغتان.

## ملاحظات الإدارة

- حذف قسم يحذف القوائم والقنوات التابعة له بعد رسالة تأكيد واضحة.
- حذف قائمة يحذف القنوات التابعة لها بعد التأكيد.
- كتالوج القنوات مستقل تماماً عن قنوات المباريات القديمة.
- الصور المقبولة: JPG وPNG وWebP وGIF، وبحجم أقصى 4MB حسب الحماية الموجودة مسبقاً.
