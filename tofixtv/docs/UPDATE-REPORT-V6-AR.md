# تقرير تحديث واجهة القنوات — V6

## المنفذ

- إزالة البانر النصي الكبير من أعلى صفحة القنوات واستبداله بسلايدر صور اختياري.
- إدارة السلايدر من لوحة التحكم: صورة فقط، فتح قائمة قنوات، أو فتح رابط إعلان.
- عرض قوائم القنوات وبطاقات القنوات بنسبة البوستر 2:3 وبنفس عنوان وتدرج بطاقات الأفلام.
- عرض قناتين في الصف على الجوال بدلاً من ثلاث بطاقات صغيرة.
- تقليل المسافة بين الهيدر وبداية المحتوى وبين عنوان المجموعة والبطاقات.
- إبقاء صورة القناة ممتدة داخل مساحة البوستر مع محاذاة ثابتة.
- جعل المفضلة اختيارية بالكامل؛ لا تتفعّل إلا بعد ضغط المستخدم وتبقى محفوظة محلياً على جهازه.
- إبقاء فتح القنوات عبر `xmtv` والحزمة `com.aloka.live.app` ورابط التنزيل `https://t.me/alokalive`.

## فحوصات السلامة

- اجتاز JavaScript فحص البنية وإعادة التصغير.
- اجتاز JSON فحص البنية.
- اجتاز 155 ملف PHP فحص المحلل.
- بقيت ملفات `AppChannels` و`ChannelLib` و`Streams` و`AppLinks` و`helpers.php` و`config.php` متطابقة مع V5.
