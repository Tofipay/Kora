# تقرير تحديث ALOKA Live — الإصدار V8

## الإضافات المنفذة

- إضافة تشفير وفك تشفير رابط القناة من نافذة القناة في لوحة الإدارة.
- التشفير مطابق لـ AES-192-ECB مع PKCS7 ومخرجات Base64 المستخدمة في التطبيق.
- حفظ حالة الرابط المشفر لمنع التشفير المكرر وإظهار زر فك التشفير عند التعديل.
- تغيير رابط قناة تيليجرام في الموقع والمساعد إلى `https://t.me/alokalive`.
- إضافة رابط تنزيل اختياري لكل فيلم أو مسلسل من لوحة الإدارة.
- إظهار زر التنزيل للمستخدم فقط عند حفظ رابط HTTP/HTTPS صالح وكان المحتوى متاحاً للمشاهدة.
- إظهار شعار الموقع داخل مساحة الصورة أثناء تحميل الصورة الأصلية، مع بقائه بديلاً عند فشل التحميل.

## الضمانات

- لم تتغير ملفات `AppChannels` و`ChannelLib` و`Streams` و`AppLinks`.
- لم تتغير دالة إنشاء `intent` أو الحزمة `com.aloka.live.app` أو Scheme `xmtv`.
- بقي `User-Agent: com.aloka.live.app` كما هو.
- لم تُحذف أي ميزة سابقة.

## الفحوصات

- تحليل بنية جميع ملفات PHP.
- فحص JavaScript الأصلي والمصغّر.
- فحص جميع ملفات JSON وWeb Manifest.
- اختبار تطابق تشفير AES مع CryptoJS 4.2.0 واختبار فك التشفير إلى النص الأصلي.
- إعادة بناء ملفات CSS وJavaScript المصغّرة.
